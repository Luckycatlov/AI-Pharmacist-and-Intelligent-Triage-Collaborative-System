## 🚀 立即可用的部署方式

### 方式1：本地LangServe（推荐新手）⭐
```bash
# 一键启动
python deploy.py

# 访问文档
http://localhost:8000/docs
```
**优势**: 简单快速，便于开发测试

### 方式2：Docker容器（推荐生产）⭐⭐
```bash
# 构建镜像
docker build -t medical-qa-system .

# 运行容器
docker run -d -p 8000:8000 medical-qa-system
```
**优势**: 环境一致，易于部署

### 方式3：Docker Compose（推荐团队）⭐⭐⭐
```bash
# 一键启动
docker-compose up -d

# 查看状态
docker-compose ps
```
**优势**: 完整环境，编排管理
